# MGTA495S Course Website

This repository contains my Quarto website for **MGTA495S Marketing Analytics (Spring 2026)**.

The site is intended to serve as a **course-facing website** for presenting selected work in a structured web format. It is designed for academic presentation purposes and website development practice within the course context.

## Structure

The repository includes the source files for the Quarto website together with the rendered output used for deployment through GitHub Pages.

Typical contents may include:

* Quarto source pages (`.qmd`)
* site configuration (`_quarto.yml`)
* styling files such as `styles.css`
* static assets such as images
* rendered website files in `docs/`

## Publishing

The website is rendered locally with Quarto and published through **GitHub Pages** using the `docs/` folder.

## Academic Use

This repository is intended as a **personal course website** for presenting selected materials in website format. It is **not intended as a public archive of reusable assignment solutions or course answer keys**. Content included here is limited to materials that are appropriate for public-facing academic presentation.
